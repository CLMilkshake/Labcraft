The History of Labcraft

Labcraft has been around for 2 years, and has been various different things. We've been trying to keep worlds around so we have records of everything. This is a small timeline of everything. You can find a museum opening soon on the server.

Bedrock World (Around 2019)

Back during this time, CLMilkshake didn't have the Java Edtion of Minecraft, but he saw the YouTubers playing on these big servers, so he decided to make one. This first world was called "CLJailbreak", trying to imitate prison servers. You would mine blocks and earn money to develop a skyblock island with spawners, and blocks.

The First SMP (Late 2019)

Labcraft was purchased and the first thing it became was an SMP. Members included sir1toasts1alot, Computer_Error_, BudGum, MrBoogBear, and sometimes a few others. We don't have much on this because it seems the world wasn't saved, but everyone lived around spawn, and there were multi colored villager statues. The following picture is a remake.

The Second SMP (Mid to Late 2020)

Eventually it was decided to reset the server to give everyone more to work towards again. Now that everyone had at least decent skill, this SMP had a lot more to offer. This is also the SMP that the video "The Flying Toast" took place, and a few others too that we can't find anywhere yet. This SMP featured locations such as the first prison, Milkshake & Master's Lab, and more. We do have a copy of this world, and some of it's items will be in the museum.

Series of SMPs (Late 2020 to 2021)

After about 6 months of the last SMP, the server was constantly reset. We have a few of these worlds, but not very many. We don't have a ton of info still, but not much happened anyways.

The Tournament (Late 2021)

This was around the time that an event called MCC, (Minecraft Championships), would sweep the attention of the server members. CLMilkshake saw this and once again wanted to do it, so he tried making it. It was not the best... As far as we know we don't have a copy of this world, but we have a short video showing part of the tournament being played.

Everything Server (Late 2021)

This was when Labcraft was givin other games besides the ongoing SMP, it had Skywars, and Bedwars, with more minigames coming. It also had a few other parts to it. This was also about the time that the other server CLMilkshake managed came up. It was a bigger server, so everyone moved the SMP to there to avoid lag, but we will still include those SMPs here.

The Edge SMP (Late 2021 to 2022)

This server hosted many SMPs, and sometimes minigames. This was also around the time that zaxs277 joined the group. This server was bigger, so it could handle more. There was once 3 SMPs running at the same time. We have a few of these worlds saved too. This server became the main server for the group, leaving Labcraft as either unused or an event server.

CABMO Event (Mid 2022)

This was the first event that CLMilkshake made that he took more seriously. He worked with his team of staff, Computer_Error_, sir1toasts1alot, and zaxs277, with some help from Fin_is_out, to start making another event. This was never completed, and was missing any working games. We have a copy of this world too.

Labcraft Tournament (Now)

We took some aspects from CABMO and upgraded them to the new tournament. We have put a lot of work into these games, and hope you enjoy this version of Labcraft. While we all miss the OG days, there are still good days to come (hopefully).
